* RPC与REST的区别

  |      | 耦合性                                 | 消息协议        | 通信协议 | 性能        | 案例                  | 对外开放                        |
  | ---- | -------------------------------------- | --------------- | -------- | ----------- | --------------------- | ------------------------------- |
  | RPC  | 强耦合，服务端与客户端必须以特定的格式 | 二进制          | TCP      | 高          | Dubbo、thrift         | 对外一般需要转换成REST/文本协议 |
  | REST | 松散耦合                               | 文本，XML、JSON | HTTP     | 一般低于RPC | SpringMVC、SpringBoot | 直接可以对外开放                |
